# is-it-down web

Next.js 16 dashboard frontend (App Router) using shadcn-style UI components.

## Run locally

1. Install dependencies:

```bash
bun install
```

2. Set API base URL:

```bash
cp .env.example .env.local
```

3. Start dev server:

```bash
bun dev
```

The app expects the backend API to be reachable at `API_BASE_URL` (default `http://localhost:8080`).
